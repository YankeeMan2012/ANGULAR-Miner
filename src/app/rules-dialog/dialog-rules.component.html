Game rules
  • The number in a cell shows how many mines are hidden around a given cell. This number will help you understand where the safe cells are and where the mines are.
  • If there is an empty cell next to an open cell, it will open automatically.
  • If you open a cell with a mine, then the game is lost.
  • To mark a cell containing a mine, right-click it.
  • Once you have marked all the mines, you can move the cursor over an open cell and press the right and left mouse buttons at the same time. Then all free cells around it will open.
  • The game continues until you open all cells without mines.
